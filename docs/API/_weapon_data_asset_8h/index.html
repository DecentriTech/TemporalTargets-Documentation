📄 File: WeaponDataAsset.h

Defines the weapon data asset used to configure gameplay parameters.

Details

📝 Detailed Description This file declares the UWeaponDataAsset class, a configurable Unreal Engine DataAsset that drives weapon behavior by exposing tunable properties for:

  • Base weapon metadata such as name, damage, and fire rate.

  • Hitscan configuration like maximum trace distance.

  • Ammo handling, including magazine size, reserve capacity, reload rules, and infinite ammo flags.

  • Spread system including minimum/maximum spread and movement-specific modifiers.

  • Recoil tuning for bullet direction and visual effects (camera kick, FOV punch).

  • Pattern-based recoil behavior via a 2D spray array.

  • Blueprint integration via BlueprintReadOnly and BlueprintReadWrite attributes.

This asset is designed to support both visual feedback systems and gameplay balance tuning.

Gameplay Data Assets

📘 Class UWeaponDataAsset A data asset that centralizes all configurable weapon parameters.

This asset drives the weapon behavior in code by exposing:

  • WeaponName for UI and logging.

  • Damage and DamageType for hit resolution.

  • FireRate controlling rounds per minute.

  • Ammo management via MagSize and MaxReserveAmmo .

  • ReloadDuration timing and Auto-Reload behavior.

  • Spread parameters such as MinSpread , MaxSpread , and recovery via SpreadRecoveryRate .

  • Recoil scaling with BulletPatternMultiplier and CameraRecoilMultiplier .

  • Visual recoil effect through FOVPunchAmount .

  • Custom spray pattern via SprayPattern array.
