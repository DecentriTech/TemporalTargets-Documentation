ğŸ“ Detailed Description This file defines the ATACharacter class, which is the main player-controlled character. It implements core gameplay interfaces and systems:

  • Interaction via IBPI_Interactable

  • Damage handling via IDamageableInterface

  • Input binding for combat and interaction using the Enhanced Input System

  • Weapon handling and inventory management via dedicated components

  • Ghost replay recording with UReplayRecorderComponent

  • Real-time combat behavior with UWeaponComponent
