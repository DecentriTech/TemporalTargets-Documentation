ğŸ“„ File: FPlayerFrameData.h

Defines data structures for capturing and replaying player frame states, interactions, and combat events.

Details

ğŸ“ Detailed Description This file contains core replay-related types used to:

  • Record per-frame player state including movement, rotation, and actions.

  • Log weapon shot events and interactions for deterministic playback.

  • Manage grouped frame data for ghost replays.

Key types include:

  • EReplayActionFlags: Bitmask for player actions per frame.

  • FRecordedShot: Captures shot metadata during a frame.

  • FPlayerFrameData: Encapsulates complete per-frame player state.

  • FReplayAttempt: Manages ownership of replay data with unique IDs.
