Details

ğŸ“ Detailed Description This file defines the UReplayRecorderComponent class, which is responsible for:

  • Capturing frame-by-frame player data at a specified interval.

  • Logging interaction and combat events during gameplay.

  • Exposing recorded replay data for later playback or analysis.

Modular Components

ğŸ“˜ Class UReplayRecorderComponent Component that captures and stores per-frame player state for replay.

UReplayRecorderComponent ticks every frame and, when active, collects a stream of FPlayerFrameData at intervals defined by RecordingInterval. It also records interaction events via CaptureInteractionData() and shot events via CaptureShotData(). All recorded frames are exposed through GetRecordedData().
