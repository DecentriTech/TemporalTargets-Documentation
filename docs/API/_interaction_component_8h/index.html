Declares the interaction logic component for player-focused actor interaction.

Details

ğŸ“ Detailed Description This file defines the UInteractionComponent class, responsible for:

  • Detecting interactable actors using a capsule sweep.

  • Managing focus state for nearby interactables.

  • Handling the interaction trigger logic for focused actors.

The component is designed to work with actors implementing UBPI_Interactable and integrates with ghost control systems via the bIsGhostControlled flag.

Modular Components

ğŸ“˜ Class UInteractionComponent Detects and handles player interactions with interactable actors.

Uses a capsule sweep (radius = CapsuleRadius, half-height = CapsuleHalfHeight) out to MaxInteractDistance to focus on actors implementing UBPI_Interactable. When focus changes it fires OnFocusedByInteractor and OnUnfocusedByInteractor on ABaseInteractableActor. Actual interaction happens via Interact().
